1.1 - Coding 中启用任务故事点和工时估算和自定义流程

背景：

  1. 在研发管理过程中需要对每个任务进行时间点估算，用以掌握迭代下的工作进度。
  2. 需定义每个任务的工时估算
  3. 在需求和任务管理流程中需要定义需求和任务的工作流为 未开始-处理中-已完成-已审核，其中前三个步骤处理人为被分配该任务的人员，如处理人接件后可按照工作进度自行更改任务的状态直至“已完成",”已审核“为该任务在变更为”已完成“后被审核人确认后更改为此状态，至此，该任务完成并审核。

操作：

  1. 使用故事点 https://coding.net/help/docs/collaboration/pattern/scrum/story-points.html#use 迭代统计 https://coding.net/help/docs/collaboration/pattern/scrum/statistics.html

  2. 任务工时估算 注：敏捷项目管理迭代状态中只能查看故事点燃尽图。

  3. 自定义工作流 https://coding.net/help/docs/collaboration/configuration/workflow.html#project

1.4 - coding中创建项目管理流程模板应用于项目

背景：研发项目经理提出需要在所有研发项目中增加审核流程，需要针对此需求在团队中创建项目配置方案并应用于此类研发项目中。 步骤：

  1. 团队设置中心 – 功能设置 – 项目协同 – 配置方案
  2. 配置模板工作流增加审核流程
  3. 配置模板中的事项类型（双击项目配置方案），增加预估工时属性和工时记录属性
  4. 在研发项目中使用该项目配置方案 项目协同 – 同步配置方案（将定义好的方案应用于此项目）
  5. 查看研发项目任务状态已经出现审核状态

2.2 - Research on Password Management of KTW Delivery Project System

调研背景：

  • 在目前进行的云原生KTW项目中，涉及多人协作完成平台基础设施的搭建，KTWimage的部署和应用部署配置工作，其中，尽管最大限度的采用账号密码和登录操作者对应，但系统的登录账号密码需要在相关人员中共享使用，为了确保系统账号密码的安全，需要调研可行的账号密码保存分享方式，使得: 1.账号密码安全存储 2.账号密码安全的共享 3.账号密码的访问便于管理

腾讯云数据保险箱

  • 数据保险箱 产品概述-产品简介-文档中心-腾讯云 (tencent.com)
  • 数据保险箱（Cloud Data Coffer Service，CDCS）是腾讯云为用户提供的更高安全系数的数据加密存储服务，用于满足客户安全且精细化的数据安全管理需求，符合监管和合规要求。
  • 数据保险箱目前正在内测中，还未开放线上自助购买，如果您想要了解更多产品信息，可前往 数据保险箱详情页 查看。
  • 数据保险箱的主要应用场景包括：敏感文件保护和权限控制。支持控制台和API。
  • 主要用户账户密码信息的加密存储管理。

腾讯云凭据管理系统

  • 凭据管理系统简介_凭据管理系统购买指南_凭据管理系统操作指南-腾讯云 (tencent.com)
  • 凭据管理系统（Secrets Manager，SSM）为用户提供凭据的创建、检索、更新、删除等全生命周期的管理服务，结合资源级角色授权及全面细致的审计管控，轻松实现对敏感凭据的统一管理。用户或应用程序可通过调用凭据管理系统 API，规避敏感配置及敏感凭据硬编码等风险问题，同时可有效避免敏感信息泄密以及权限失控带来的业务风险。
  • 可在自定义凭据中添加查看编辑凭据信息。

腾讯云密钥管理系统

  • 密钥管理系统 产品概述-产品简介-文档中心-腾讯云 (tencent.com)
  • 目前需求主要是对系统的登录账号和密码做安全管理，其白盒密钥产品需一次性支付￥20,000开通使用。
